# Abhay's Hackpad Macropad
My personal macropad for everyday use. 

Bill of Materials: <br>
- 1x OLED Screen
- 1x 3D Printed Case
- 6x Cherry MX Switches<br>
- 2x SK6812 MINI Leds<br>
- 1x XIAO RP2040<br>
- 6x Blank DSA Keycaps<br>
- 4x M3x16 Bolt<br>
- 4x M3 Heatset<br>
